# Real public-domain shanties — download & drop in here

**Right-click → Save As** each link into THIS folder
(`game/audio/shanties/`) using the **target filename** shown. The game auto-loads
them on next reload (console logs each one).

You don't need all of them — even just `drunken-sailor.mp3` + `blow-the-man-down.mp3`
is enough. More = a varied playlist.

## More sources to browse for additional PD shanties
- Internet Archive (search "sea shanty", filter Public Domain): https://archive.org
- LoC / IA "Sea Shanties" episode (Blow the Man Down, Haul the Bowline, etc.):
  https://archive.org/details/20210129-sea-shanties
- Wikimedia Commons category: https://commons.wikimedia.org/wiki/Category:Audio_files_of_sea_shanties
- The Public Domain Review sheet-music collection: https://publicdomainreview.org/collection/sea-shanties/

## How the game uses them
- `drunken-sailor` / `haul-away-joe` / `rio-grande` / `a-roving` → **sailing** music (playlist)
- `blow-the-man-down` → **battle** music
- `shenandoah` → **menu / cove**
- Anything missing falls back to the built-in synth.
Hard-reload after adding; check the console (F12) for "[shanties] loaded …".
